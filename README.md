## Como executar:

* Clone este repositório.

### Instale as bibliotecas necessárias pelo terminal

*`npm install -g npm`
*`npm install react-router-dom`
* `npm install swiper`

### Inicie o projeto

* No terminal, digite `npm start` e abrirá uma página no navegador.
